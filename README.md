# CodeLingo

An interactive learning platform for mastering **Python, JavaScript, Java, and HTML/CSS** through mini-lessons, quizzes, daily challenges, AI-powered answer checking, and engaging mini-games.

---

## üöÄ Features

- **User Authentication**  
  Registration, login, and persistent user sessions.

- **Interactive Lessons**  
  Bite-sized lessons for Python, JS, Java, and HTML/CSS.

- **Quizzes & AI-Verified Answers**  
  Google AI Studio API is used to evaluate open-ended answers.

- **Daily Challenges**  
  Encourages consistent learning and streak building.

- **User Progress Tracking**  
  XP, levels, completed lessons, and achievements.

- **Themes**  
  Light, Dark, and Pink themes.

- **Avatar Shop**  
  Users can purchase avatars using in-app currency.

- **Leaderboard**  
  Global ranking among all learners.

- **Mini-Games**
  - **BugInfection**
  - **MemoryCode**
  - **RefactorRush**
  - **VariableHunt**
